目录结构

微信小程序中，最关键也是必不可少的，是 app.js、app.json、app.wxss 这三个。

  1. 文件后缀
    1. js后缀：脚本文件
    2. json后缀：配置文件
    3. wxss后缀：样式表文件
    4. wxml后缀的文件是页面结构文件。
  2. 程序启动
  3. app.js是小程序的脚本代码。我们可以在这个文件中监听并处理小程序的生命周期函数、声明全局变量。调用框架提供的丰富的API。
  4. app.json 是对整个小程序的全局配置。我们可以在这个文件中配置小程序是由哪些页面组成，配置小程序的窗口背景色，配置导航条样式，配置默认标题。
  5. 微信小程序中的每一个页面的【路径+页面名】都需要写在 app.json的pages中，且pages中的第一个页面是小程序的首页。
  6. app.wxss是整个小程序的公共样式表。我们可以在页面组件的class属性上直接使用app.wxss中声明的样式规则。
  7. 每一个小程序页面是由同路径下同名的四个不同后缀文件的组成，如：index.js、index.wxml、index.wxss、index.json。
